# 🏠 Housing Price Prediction using Regression

## 📌 Project Overview
This project predicts housing prices using a Machine Learning regression model based on the Boston Housing dataset. It demonstrates the complete workflow from data preprocessing to prediction.

## 📊 Dataset
- Source: Kaggle Boston Housing Dataset  
- Contains features like crime rate, number of rooms, tax rate, etc.  
- Target variable: `MEDV` (Median house price)

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

## 🚀 Steps Performed
1. Data Loading  
2. Data Preprocessing  
3. Train-Test Split  
4. Model Training using Linear Regression  
5. Model Evaluation (Mean Squared Error)  
6. Prediction of new house prices  

## 📈 Model
- Algorithm Used: Linear Regression  
- Evaluation Metric: Mean Squared Error (MSE)
